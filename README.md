# Voice-Gender-Recognition-System

An AI-based real-time voice gender recognition system developed using Python and Machine Learning.

The system records live voice input, extracts audio features, predicts gender, and automatically saves voice samples into dataset folders.

**Features**

* Real-time voice recording
 *Gender prediction using ML model
* Automatic voice sample saving
*Audio feature extraction
 *Dataset organization
 *Python-based implementation

 **#Technologies Used**
- Python
- Machine Learning
- Scikit-learn
- Librosa
- SoundDevice
- SoundFile
- NumPy

Install Dependencies
